Run it in four steps

  1. Install. pip install -r requirements.txt
  2. Start the server. python -m robodog.server --host 0.0.0.0 --port 8770 (on Linux: sudo systemctl enable --now bluetooth first).
  3. Open the dashboard. Visit http://<host>:8770/, power the dog on, close the phone app.
  4. Scan → Connect → drive. If the dog isn't listed by name, hit Scan all and pick the strongest nearby device.
from robodog import RobodogBLE
dog = RobodogBLE()
await dog.connect()
await dog.send_command('sit')
await dog.send_command('dance')
await dog.disconnect()
